关键差距总结

❌ 群聊缺失
  • Markdown 渲染（仅 <br> 替换）
  • 流式输出显示
  • 入场动画
  • 消息操作（复制/导出）
  • 长文本折叠/展开
  • 自动滚动
  • 时间分隔线
✅ 群聊独有
  • 多 Agent 彩色头像 + 名称
  • 发言人颜色标识（左边框）
  • 不确定性标记高亮
  • 审核检查点面板
  • 预算进度条
  • 思考过程折叠

🔴 发现的逻辑 BUG

审核间隔导致无限循环：当 round % reviewInterval === 0 时触发 continue，但 round++ 在 continue 之后，导致同一轮反复触发审核，永远无法前进。
