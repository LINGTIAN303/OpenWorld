整合所有已确认的设计决策，请审阅后确认

📋 设计决策总览

目标用户 两者兼顾 — 默认对非技术用户友好，提供高级模式
模式切换 全局开关 + 节点级可覆盖
空白画布引导 智能建议输入框 + 模板库
属性面板 简化模式（3-4 字段/中文标签/下拉）↔ 高级模式（全字段/原始键名）
节点面板 分组归类 + 搜索框 + 可折叠

🎨 改动清单（7 大模块）

1. 空白画布引导 — OnboardingOverlay

  • 画布为空时显示引导覆盖层
  • 中央「描述你想自动化的任务」输入框 → 触发 AI 生成骨架
  • 底部模板卡片横排：邮件处理 / 内容创作 / 数据分析 / 定时任务
  • 点击模板 → 加载预定义工作流到画布
  • 添加第一个节点后引导自动消失

2. 全局模式切换 — EditorModeToggle

  • 工具栏右侧添加 🌙/⚙️ 切换按钮
  • 简化模式：所有属性面板显示简化表单
  • 高级模式：所有属性面板显示完整字段
  • 状态持久化到 localStorage

3. 属性面板双模式 — PropertyPanel 重构

  • 每个节点类型定义 simplifiedFields 列表（3-4 个最常用字段）
  • 简化模式：中文标签 + 下拉/输入 + 占位提示 + 隐藏可选字段
  • 高级模式：原始键名 + 完整类型控件 + 全部字段
  • 节点级覆盖：面板右上角「高级 ▸」/「◂ 简化」按钮
  • 错误提示人性化：「请填写条件表达式」而非「缺少 config.expression」

4. 节点面板重构 — NodePanel 分组

  • 顶部搜索框，实时过滤节点
  • 4 个可折叠分组：流程控制 / 智能节点 / 循环与并行 / 高级
  • 每个节点卡片：彩色左边框 + 中文名 + 图标 + 一句话描述
  • 拖拽到画布添加节点（保留现有功能）

5. 画布节点视觉升级 — WorkflowNode 重设计

  • 节点显示中文名（简化模式）或英文类型名（高级模式）
  • 节点状态指示：运行中（脉冲动画）、成功（✓ 绿色边框）、失败（✕ 红色边框）
  • 连接线标签中文化：「是/否」而非「true/false」
  • 悬停节点显示简短描述 tooltip

6. 连接交互优化

  • 连接时显示语义标签建议（条件节点 → 「是」/「否」端口）
  • 无效连接实时提示（如 start→start）
  • 选中连接线高亮 + 显示删除按钮

7. 校验与反馈人性化

  • 校验错误用中文 + 图标提示：「⚠️ 工作流需要一个开始节点」
  • 保存成功提示：「✅ 工作流已保存」
  • 运行前自动校验，错误高亮对应节点
