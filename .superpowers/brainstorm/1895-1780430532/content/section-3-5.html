💡 关键设计:决策弹窗是阻断式(运行暂停 + 计时器可见),但非模态阻塞——用户可以"稍后再说"延后,后台照常计时;超时自动 fallback 到节点配置里的默认分支;
所有决策事件写入 run 详情日志(用 WsCodeBlock 显示),便于复盘;决策本身的 prompt + 上下文 + 选项 + 选择 + 时间戳入库,后续可做"哪些决策经常被选"的统计。
