📊 数据模型扩展

新增 Store: mindmapStore
interface MindmapState {
  // 嵌套图导航
  navigationStack: string[]        // 实体ID栈 ["global", "char-123", ...]
  currentRootId: string | null     // 当前根实体ID，null=全局图
  
  // Section 分组
  sections: Map<string, Section>   // sectionId → Section
  
  // 画布状态
  customNodes: Map<string, CustomNode>  // 自由画布上的自定义节点
  edgeStyleOverrides: Map<string, EdgeStyle>  // 单条边样式覆盖
  
  // 键盘流
  keySequence: string              // 当前按键序列缓冲
  keySequenceTimer: number | null  // 序列超时定时器
  
  // 详情侧栏
  detailPanelVisible: boolean
  detailPanelAutoMode: boolean     // 是否处于自动跟随模式
}

interface Section {
  id: string
  name: string
  color: string
  childNodeIds: string[]
  isEntity: boolean                // 是否已升级为实体
  entityId?: string                // 关联的实体ID
  bounds: { x: number; y: number; w: number; h: number }
}

interface CustomNode {
  id: string
  parentId: string                 // 所属实体画布 (嵌套图)
  type: 'textbox' | 'image' | 'note' | 'link'
  name: string
  content: string
  position: { x: number; y: number }
  style: Record<string, any>
}

interface EdgeStyle {
  lineStyle: 'bezier' | 'straight' | 'taxi' | 'orthogonal'
}

🔄 嵌套图数据流

用户 Ctrl+双击节点
  → mindmapStore.navigationStack.push(nodeId)
  → mindmapStore.currentRootId = nodeId
  → buildElements() 重新计算:
      - 从 relationStore 获取该实体的所有直接关系
      - 生成以该实体为中心的子图节点和边
      - 从 customNodes 加载该实体画布上的自由节点
  → Cytoscape 重新渲染
  → 面包屑更新: 全局图 > 张三

用户 Esc / 面包屑点击
  → mindmapStore.navigationStack.pop()
  → mindmapStore.currentRootId = stack[top]
  → buildElements() 重新计算 (回到上层或全局)
  → Cytoscape 重新渲染

⌨️ 键盘流实现

全局 keydown 监听 → 
  if (单键基础操作):
    Delete → 删除选中
    Escape → 取消/返回上层
    I → 切换详情侧栏
    F → 适应视图
    Space+拖拽 → 平移画布
  
  else (按键序列):
    追加到 keySequence 缓冲
    重置 500ms 超时定时器
    if (匹配已知序列):
      执行对应操作
      清空缓冲
    else if (超时):
      清空缓冲
      显示提示 "未知序列: xx"

左下角 MindmapKeyHint 显示:
  - 当前按键序列状态 (如 "n_" 等待第二键)
  - 可用的后续键提示
