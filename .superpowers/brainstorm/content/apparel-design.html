🧥 Apparel 服饰/装备插件 — 设计方案

定位：所有可穿戴在角色身上的物品，与 weapon 构成攻防对

1. 基本信息

项目值
插件 IDofficial.apparel
插件名称服饰 / 装备
描述管理角色的服装、护甲与饰品
实体类型apparel（统一类型）
图标名apparel

2. 实体字段设计

共 12 个字段，分四组

分组字段类型选项/说明
基础apparelTypeselect上衣、下装、连衣裙、外套/披风、头饰、鞋履、手套、饰品/首饰、轻甲、中甲、重甲、盾牌、法袍、套装
armorClassselect无防护、布甲、皮甲、锁甲、板甲、法袍、饰品
materialtext材质描述（如：丝绸、钢铁、龙鳞）
colortext颜色
styleselect朴素、华丽、异域、军用、仪式、日常、伪装、工装
性能defensenumber防御值
weightselect极轻、轻、中等、重、极重
durabilityselect易损、普通、耐用、坚固、不毁
背景originentityRef → region产地/来源地
eratext所属时代
conditionselect完好、轻微磨损、破损、已修复、已毁、遗失
描述significancetextarea意义/背景故事

3. 关系类型设计

4 个专有关系 + 3 个跨插件复用关系

关系源→目标有向属性
worn_by 被穿着apparel → character✅since（开始穿着时间）
enchanted_with 被附魔apparel → magic✅spell（附魔效果）
manufactured_by 由...制造apparel → character/org✅—
related_apparel 关联服饰apparel ↔ apparel❌relation (同一套/成套/相克/升级)
— 以下复用已有关系 —
located_at 位于apparel → region✅(复用)
owned_by 被拥有apparel → character✅(复用)
created_by 由...制造apparel → character/org✅(复用)

4. UI 组件结构

参考 ItemView.vue 模式，细节定制化

ApparelView.vue
├── Toolbar: 搜索 + 类型筛 + 护甲等级筛 + 新建 + 批量删除

│   └── 卡片: 图标|护甲等级徽章 → 名称 → 材质 → 防御值 → 部位标签

│   ├── 基础信息区: 头像|名称|护甲等级|编辑切换

│   ├── Tab: 穿着者/存放地/关联服饰/使用记录
│   └── UniversalRelationPanel
└── EntityFormModal (新建/编辑弹窗)
