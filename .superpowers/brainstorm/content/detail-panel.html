侧栏交互模式

A

自动弹出 — 选中即显示

点击节点时侧栏自动滑出，点击空白处自动收起。最直觉的交互，但频繁切换时可能干扰。

B

手动切换 — 快捷键/按钮控制

按 I 键或点击工具栏按钮切换侧栏显隐。选中节点时侧栏内容更新但不自动弹出。用户掌控节奏。

C

智能模式 — 首次自动，之后跟随

首次选中节点时自动弹出侧栏，之后切换节点时内容跟随更新。用户可手动关闭，关闭后不再自动弹出直到再次按 I 键开启。

侧栏内容范围

侧栏应该展示多少信息？

A

精简模式 — 名称+类型+关系列表

只展示核心信息，点击"编辑"打开完整表单。侧栏窄（240px），画布空间最大化。

B

完整模式 — 所有字段+关系+内联编辑

展示实体的所有字段、描述、标签、关系列表，支持直接在侧栏内编辑。侧栏较宽（320px），类似 VS Code 的侧栏。

C

自适应 — 可折叠分区，默认精简可展开

默认展示名称+类型+关系列表（240px），各分区可展开查看详情/编辑。用户按需展开，兼顾空间和信息量。
