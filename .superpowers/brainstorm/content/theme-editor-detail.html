⚙️ 专业模式 — 高级定制能力

🔍 WCAG 对比度检测
实时检测文字与背景的对比度比值，确保可读性。不达标时显示警告，并建议替代色值。

✓ AA 4.5:1 通过
⚠ AA 3.2:1 不达标
✕ AA 1.8:1 严重不足
🔗 令牌依赖关系图
可视化展示令牌间的引用链。修改一个原始令牌时，高亮所有受影响的语义令牌和组件，避免意外破坏。

--blue-500 → --color-primary → --button-bg
                       → --input-focus
🌓 暗色/亮色同步编辑
一个主题包含暗色和亮色两套令牌映射。左右分栏同步编辑，确保两种模式下视觉一致性。切换时自动生成对应版本。
🧩 组件级令牌覆盖
全局令牌之外，可对单个组件单独覆盖样式。例如：所有卡片用 12px 圆角，但工作流节点用 4px 锐角。

全局 --radius-card: 12px
覆盖 --workflow-node-radius: 4px
🎭 动效曲线编辑器
可视化贝塞尔曲线编辑器，调整过渡和动画的缓动函数。预设弹簧/弹跳/流畅等曲线，也可自定义控制点。

cubic-bezier(0.34, 1.56, 0.64, 1)
📐 间距/字号可视化标尺
编辑间距和字号时，预览区显示实际像素标尺。拖拽调整，实时看到元素间距变化。支持网格对齐辅助线。

💻 代码模式 — 开发者终极控制

📝 JSON 直接编辑
带语法高亮和实时校验的 JSON 编辑器。直接修改主题定义文件，支持自动补全令牌名称。
💉 CSS 片段注入
在令牌系统之外注入自定义 CSS。用于实现令牌无法表达的复杂效果（动画关键帧、伪元素、特殊布局）。
📦 导入/导出/分享
导出为 .ws-theme.json 文件，导入他人分享的主题。未来可建社区主题市场。
🔄 版本差异对比
主题修改历史记录，可对比任意两个版本的差异。支持回滚到历史版本。
🧪 沙盒预览
在独立沙盒中预览主题效果，不影响当前工作环境。确认满意后再应用。
🔧 令牌公式
令牌值支持表达式：如 color-mix(in srgb, var(--color-primary) 80%, white) 或 calc(var(--space-4) * 1.5)
