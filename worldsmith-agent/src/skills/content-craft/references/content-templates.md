# 内容创作模板

## 角色模板
```
名称: [角色名]
类型: character
描述: [50-100字的角色概述]
属性:
  - 性格: [3-5个关键词]
  - 背景: [简要背景]
  - 目标: [核心动机]
  - 弱点: [性格缺陷]
```

## 区域模板
```
名称: [区域名]
类型: region
描述: [80-150字的区域描述，含地理/气候/文化]
属性:
  - 气候: [气候类型]
  - 地形: [地形特征]
  - 资源: [主要资源]
  - 人口: [大致人口]
```

## 组织模板
```
名称: [组织名]
类型: organization
描述: [60-120字的组织概述]
属性:
  - 目标: [组织使命]
  - 结构: [组织架构]
  - 领袖: [领导者]
  - 成员数: [大致人数]
```

## 事件模板
```
名称: [事件名]
类型: event
描述: [80-150字的事件描述，含起因/经过/影响]
属性:
  - 时间: [发生时间]
  - 地点: [发生地点]
  - 参与者: [关键参与者]
  - 影响: [长远影响]
```

## 物品模板
```
名称: [物品名]
类型: item
描述: [50-100字的物品描述]
属性:
  - 类型: [武器/防具/饰品/消耗品/其他]
  - 稀有度: [普通/稀有/史诗/传说]
  - 效果: [特殊效果]
  - 历史: [物品来历]
```
