# 搜索引擎使用指南

## 支持的搜索引擎

| 引擎 | 特点 | 适用场景 |
|------|------|----------|
| bing | 默认引擎，覆盖面广 | 通用搜索 |
| google | 结果质量高 | 技术资料、学术搜索 |

## 搜索策略

### 单次搜索
```
web_search({ query: "关键词" })
```

### 多角度搜索
当单次搜索结果不够精确时，换一组关键词：
```
web_search({ query: "中世纪城堡 建筑特点" })
web_search({ query: "medieval castle architecture features" })
```

### 深度阅读
对搜索结果中最相关的页面：
```
web_fetch({ url: "https://..." })
```

## 结果评估标准

- 来源权威性（学术期刊 > 官方网站 > 博客 > 论坛）
- 时效性（标注发布日期）
- 一致性（多来源交叉验证）
