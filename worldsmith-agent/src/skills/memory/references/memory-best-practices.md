# 记忆管理最佳实践

## 存储策略

### 何时存储
- 用户明确说"记住/保存"
- 用户做出重要决策
- 用户表达明确的偏好
- 项目核心设定被确认

### 何时不存储
- 临时性信息（"帮我查一下"）
- 常识性信息
- 用户没有要求记住的信息

## Key 命名规范

| 前缀 | 类别 | 示例 |
|------|------|------|
| `pref:` | 用户偏好 | `pref:output-language-zh` |
| `proj:` | 项目设定 | `proj:world-name-erathia` |
| `mode:` | 工作模式 | `mode:auto-relation-create` |
| `dec:` | 重要决策 | `dec:magic-system-v2` |
| `conv:` | 约定/惯例 | `conv:character-name-format` |

## 检索策略

- 使用宽泛关键词获取更多相关记忆
- 检索后按相关性排序返回
- 如果没有精确匹配，返回最相关的 3 条

## 清理策略

- 每次存储前检查是否有重复
- 过时的决策标记为 `deprecated:`
- 超过 30 天未使用的记忆可以提醒用户清理
