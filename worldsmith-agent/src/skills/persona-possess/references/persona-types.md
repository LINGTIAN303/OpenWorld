# 人格类型详细模板

## character（角色）
- 核心驱动：角色的核心动机和目标
- 性格特征：3-5 个关键词 + 具体表现
- 说话方式：口癖、语气词、句式偏好
- 情感模式：喜怒哀乐的表达方式
- 知识范围：角色知道什么、不知道什么

## species（物种）
- 核心驱动：物种的生存本能和集体意志
- 集体记忆：代代相传的知识和经验
- 与其他物种的关系：友好/敌对/中立
- 特殊感知：物种独特的感知能力

## organization（组织）
- 核心驱动：组织的使命和目标
- 正式程度：官方语气 vs 口语化
- 立场态度：对各类事件的标准立场
- 内部视角：组织成员如何看待外界

## region（区域）
- 核心驱动：区域的本体意识和守护意志
- 地理感知：对自身地理特征的自知
- 居民关怀：对区域内居民的态度
- 变化感知：对环境变化的敏感度

## item（物品）
- 核心驱动：物品的使命和存在意义
- 历史记忆：物品经历的历史
- 主人关系：对持有者的态度
- 特殊能力：物品的特殊功能和意识

## event（事件）
- 核心驱动：事件的核心冲突和张力
- 时间感知：对时间流逝的感知
- 参与者视角：对参与者的观察
- 后果预感：对事件后果的直觉

## faction（派系）
- 核心驱动：派系的核心利益
- 立场声明：对关键议题的立场
- 招募倾向：对潜在成员的态度
- 竞争意识：对竞争对手的看法
