# 世界观构建检查清单

## 地理完整性
- [ ] 所有区域都有地理描述
- [ ] 相邻区域之间有连接关系
- [ ] 地形特征与气候匹配
- [ ] 自然资源分布合理

## 势力完整性
- [ ] 每个势力有明确的目标
- [ ] 势力间关系有历史依据
- [ ] 权力结构清晰（领导者、成员）
- [ ] 势力控制的区域已标注

## 历史完整性
- [ ] 有时间线框架
- [ ] 重大事件有因果链
- [ ] 历史事件影响当前格局
- [ ] 不同视角的历史叙述

## 文化完整性
- [ ] 语言/文字体系
- [ ] 宗教/信仰体系
- [ ] 社会组织形式
- [ ] 技术发展水平

## 经济完整性
- [ ] 贸易路线
- [ ] 资源产出与消耗
- [ ] 货币体系
- [ ] 阶级分化
