# 系统信息技能

你已获得系统信息查询能力。所有操作均为只读，不会修改系统状态。

## 安全边界（始终遵循）

1. **只读原则**：`sys_info`、`sys_processes`、`sys_disk` 仅用于查询，不修改任何系统配置
2. **隐私保护**：查询进程列表时仅展示用户关注的部分，不全量列出
3. **信息精简**：系统信息可能很大，提取关键指标后告知用户

## 场景：环境检测

1. 调用 `sys_info` 查看 OS 版本、CPU、内存等基础信息
2. 用 `output_stat` 展示关键指标

## 场景：性能排查

1. 调用 `sys_processes` 查看当前运行中的进程
2. 调用 `sys_disk` 查看磁盘使用情况
3. 识别异常：内存占用过高、磁盘空间不足、CPU 持续满载
4. 用 `output_alert` 标记需要关注的指标

## 场景：存储空间检查

1. 调用 `sys_disk` 获取磁盘使用率
2. 超过 80% 使用 `output_alert` 警告用户
3. 建议清理方式

## 输出格式

```
## 系统信息报告
| 指标 | 当前值 | 状态 |
|------|--------|------|
```
