# 角色推演模式

## 对话推演模式
模拟两个或多个角色之间的对话，保持角色一致性。

步骤：
1. 获取所有参与角色的完整属性
2. 设定对话场景和触发事件
3. 逐轮模拟，每轮一个角色发言
4. 确保每个角色的语气、用词、立场与其属性一致
5. 对话自然推进，避免生硬转折

## 冲突推演模式
推演角色间的冲突发展和可能的解决路径。

步骤：
1. 识别冲突双方的核心利益
2. 分析冲突升级的触发条件
3. 推演冲突的 3 个可能走向
4. 为每个走向评估对各方的影响
5. 提出最可能的解决路径

## 成长推演模式
推演角色在经历事件后的成长变化。

步骤：
1. 记录角色当前状态（属性、关系、信念）
2. 引入触发事件
3. 分析事件对角色心理的影响
4. 推演角色的适应和变化
5. 输出成长后的新状态
